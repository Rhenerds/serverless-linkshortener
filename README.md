## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Set up Environment Variables:
   Create a `.env` file (or set in Vercel):
   ```env
   # Redis Connection (Optional for local dev, defaults to localhost:6379)
   REDIS_URL=redis://...
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```
